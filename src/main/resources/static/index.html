キノコラボ

キノコラボは大阪市阿倍野区のプログラミング教室です。 子どもから大人まで、ものづくりを楽しみながらプログラミング技術を身につけることができます。 個人経営の教室ならではの柔軟なカリキュラムが特徴です。 毎回払いきりの現金制またはチケット制ですので、聞きたいことがあるときだけ、1ヶ月だけなど、お好きなタイミングで受講していただけます。 もちろん、「毎週◯曜日 △時から」というような受講も可能です。 まずは、おトクな価格でご利用いただけるお試し教室をご予約してください。

空き状況カレンダー

指導内容

当教室に決まったカリキュラムはありません。 生徒ひとりひとりの能力や学びたいものに合わせた、柔軟な指導が可能です。 ※教材購入が必要なものもあります。随時ご案内いたします。

プログラミング入門

はじめは、ブロック感覚でプログラミングができるScratchという言語を使って、ゲームや簡単なアプリなどを作ります。 その後、HTMLやCSSを使って簡単なサイト構築をしたり、レベルに応じて、javascript、Python、swift、Javaなどの言語を学習します。

Minecraft

大人気ゲームMinecraftを教材にして、レッドストーン回路で論理回路を学んだり、Scratchと合わせてプログラムを組んだりします。

電子工作

ブレッドボードやRaspberryPiを使って、LEDを光らせたりします。 レベルや年齢に応じて、基盤へのはんだ付けなどもします。

Micro:bit

小型の学習用多機能デバイスMicro:bitを使ってプログラミングを学習します。

その他必要な知識

必要に応じて、アルファベット・ローマ字・英語・座標・角度・電気回路・物理演算などの指導を行います。

タイピング練習

タイピング練習サイトを使って、タイピングの練習をします。 ※注意※ 当教室はプログラミング教室であり、従来のパソコン教室によくある「Word/Excel等Microsoft製品の使い方」「Windowsの使い方」などについては触れません

料金・コース案内

料金は受講時に現金でお持ちください。 お好きなタイミング、お好きな頻度で受講していただけます。 ジュニアコースは月2〜4回、シニアコース、エキスパートコースは月4〜6回を目安に受講してください。 受講予約、空き枠確認はこちらから 学習内容はあくまで参考です。 固定のカリキュラムにとらわれない、生徒一人ひとりに合わせた指導がキノコラボの特徴です。

ジュニアコース

1回1.5時間　2500円 小学1年生〜小学4年生までのコースです。 主にScratchを用いたプログラミングや、Micro:bitを使った学習を行います。 Minecraftを使った学習では、Mod導入やプログラミング（Scratch、Python）を利用した自動化などを学習します。

シニアコース

1回1.5時間 3000円 小学5年生〜中学3年生までのコースです。 Scratchからはじめて、実力に合わせPython、Javascriptなどにレベルアップしていきます。 HTMLやCSSを使ってWeb製作をしたり、IoT機器を使った学習も行います。 Minecraftを使った学習では、レッドストーン回路を使った動く仕掛けを作ったり、マルチサーバ構築などを学習します。

 エキスパートコース

1回2時間 5000円 高校1年生〜大人のコースです。 プログラミング言語は実力や好みに合わせて学習していきます。 Web製作、アプリ製作などをはじめ、個人のやりたいことに合わせて学習を進めます。 Minecraftを使った学習では、JavaでModをを作ったりします。
