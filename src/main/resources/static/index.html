キノコラボ

キノコラボは大阪市阿倍野区のプログラミング教室です。 子どもから大人まで、ものづくりを楽しみながらプログラミング技術を身につけることができます。 個人経営の教室ならではの柔軟なカリキュラムが特徴です。

毎回払いきりの現金制ですので、聞きたいことがあるときだけ、1ヶ月だけなど、お好きなタイミングで受講していただけます。 もちろん、「毎週◯曜日 △時から」というような受講も可能です。
